# SBTI 人格测试技能

## 项目简介

SBTI 人格测试是一个基于"发疯文学"和"互联网自嘲"风格的恶搞版 MBTI 人格测试技能。通过 30+ 道幽默风趣的题目，从 15 个维度对用户进行评分，最终匹配到 27 种罕见的"神经质"人格类型之一。

> MBTI已死，SBTI当立！

## 功能特点

- **30+ 道精心设计的题目**：涵盖自我认知、情感态度、社交风格等多个维度
- **15 维度评分系统**：科学（但不太严谨）的多维度人格分析
- **27 种独特人格类型**：包括"拿捏者"、"送钱者"、"屌丝"、"小丑"等
- **隐藏彩蛋**：特殊触发条件可解锁隐藏人格
- **毒舌风格**：全程陪伴的"发疯文学"式交互体验

## 核心算法

### 维度计分

测试包含 15 个维度，每个维度由 2 道题构成：

| 维度组 | 维度代码 | 说明 |
|--------|----------|------|
| 自我认知 | S1, S2, S3 | 自我评价、自我认知、自我追求 |
| 情感态度 | E1, E2, E3 | 信任感、恋爱观、独立性 |
| 行动风格 | A1, A2, A3 | 人性观、规则意识、目标感 |
| 成就驱动 | Ac1, Ac2, Ac3 | 成就动机、决策风格、执行力 |
| 社交模式 | So1, So2, So3 | 社交开放度、亲密度、社交面具 |

**计分规则**：
- 每题选项分值：A=1分, B=2分, C=3分
- 维度总分转换：≤3分→L(低), =4分→M(中), ≥5分→H(高)

### 匹配算法

```
距离(Distance) = Σ |用户维度值 - 标准维度值|
匹配度(Similarity) = MAX(0, ROUND((1 - Distance / 30) * 100))%
```

### 特殊触发

- **酒鬼 (DRUNK)**：爱好选择"饮酒" + 追问选择"把白酒当水喝"
- **傻乐者 (HHHH)**：与所有常规人格匹配度均 < 60%
